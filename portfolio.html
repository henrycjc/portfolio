Easitag Pty Ltd commissioned a mobile application for encoding NFC data to NFC enabled media: Easiwayv NFC (available on the Google Play store). I am also currently designing and developing a large scale propriety product for Easitag.

Technologies

Ionic Framework / AngularJS / Apache Cordova: I love using theses frameworks and technologies. Ionic is incredibly flexible and enables fast prototyping for scrum like methodologies. It helped me deliver on crucial client requirement to have the product easily ported to multiple platforms when new technology becomes available on them. AngularJS is an ingenious framework which was a pleasure to learn and make use of for the Easiwayv project.

History Chains is an academic research based social network of sorts, developed by three undergraduate students at The University of Queensland. It is targeted towards younger students, 14 to 18 years old studying Modern Australian History and other related subjects. History Chains makes extensive use of the Trove API, an initiative by the National Library of Australia to catalogue Australia's history digitally.

Technologies

PHP: My 'goto' programming language. I have been using since 2010 and I know it very well so this project was more about applying my existing skills rather than learning new ones. However, it did help to learn more about AJAX.
MySQL: DBMS of choice - I like its simplicity and number of features. Complete implementation of SQL standard is not necessary here.

DecoWorld was a game developed by myself and 200 other students for a second year design computing course at my university.

Technologies

Java: I find Java personally challenging but for large scale projects such as this the language's design shines very bright. Technically, it is not my strong suit however the problem solving aspect of the project became the greater challenge and Java's tools merely provided an outlet for solutions.
LibGDX: Feature full and very powerful graphics library. It was great to get some experience using a game-centric library even though it wasn't something like Unity or Unreal Engine.
Git: Git was a major part of this project as there ended up being 8000+ commits over 13 weeks, conflict resolution became a major aspect of the project's success.
